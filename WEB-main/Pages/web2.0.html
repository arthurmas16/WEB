WEB 2.0

Es un modelo de web que permite a los usuarios interactuar y colaborar entre sí, como creadores de contenido. La red social de este tipo ya no es entonces un simple contenedor o fuente de información, y se convierte en una plataforma de trabajo colaborativo.
Una wiki es un sitio web construido de forma colaborativa con el objetivo de crear un lugar de conocimiento. La herramienta Google Sites nos permite hacer wikis de una manera fácil: se puede hacer un sitio web con la colaboración de varios usuarios.

Los blogs permiten expresar opiniones en un sitio web y compartirlas, debatiéndolas y contrastándolas con los de otros usuarios, pero siempre bajo el control del creador del blog que, en última instancia, es el que decide lo que puede publicarse o no en él. Los blogs se crean a base de entradas que se muestran en orden cronológico inverso: las primeras entradas que se ven son las últimas entradas creadas. Es útil asignar a cada entrada una categoría según la temática de la que traten.
